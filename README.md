# Funkin' Mod Menu
A mod for vanilla Friday Night Funkin' that adds a mod menu for enabling and disabling mods without having to use File Explorer!

## Downloads
You can download the latest stable release from GameBanana: https://gamebanana.com/mods/603371
You can contribute to this repository by making pull requests to the `develop` branch.

## For Contributors:
You must make your own branch based on `develop` when making pull requests!
DO NOT PULL REQUEST TO THE ACTUAL `develop` BRANCH, IT WILL BREAK YOUR FORK!
